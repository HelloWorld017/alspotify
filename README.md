# old-lyric-viewer

> `스포티파이` 혹은 `유튜브 뮤직`등의 `뮤직 플레이어`에서 재생중인 노래의 가사를 화면에 표시해줍니다.  
> 가사가 존재하지 않는 경우, 스포티파이의 내장 가사를 표시합니다. (**스포티파이로 재생하는 경우만 해당**)  
> node.js의 Qt 바인딩인 nodegui를 사용해 제작되었습니다.

> [!WARNING]
> 이 프로그램은 제거되었습니다. 더 이상 사용하실 수 없습니다.
